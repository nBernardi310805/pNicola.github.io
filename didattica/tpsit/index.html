Come creare un sito web

Premessa:

Imparare a creare un sito web non è come imparare a disegnare, cosa che tra l'altro non so fare, ma è più complesso perché bisogna imparare i codici descrittivi (HTML, CSS) e possibilmente il JavaScript.

Per creare un sito web

L'HTML

L'HTML è un linguaggio descrittivo. Serve per creare il sito web nella sua struttura, quindi titoli, testo, immagini

L'iframe sovrastante mostra la Home Page del mio sito senza CSS o JavaScript.

cc

Il CSS

Il CSS è il linguaggio descrittivo che serve a dare una grafica al sito.

Il JavaScript

Il JavaScript serve a rendere il sito interattivo. Ha una sintassi simile a quella del C.

Perché non basta "rubare" un template fare un sito?

Perché non ha senso fare un sito basandosi solo su un template perché si è limitati dalla grafica, dal vecchio contenuto del modello, ecc.

Creazione di un sito - cose essenziali

Consigli

  • Avere un'idea.

Sembra una cosa banale e scontata, ma il mio primo consiglio sulla creazione di siti è inanzitutto avere un'idea precisa riguardo ai contenuti o allo scopo del sito. Questo primo consiglio è collegato agli altri, come vedrai tra poco.

  • Mantenere ordine

La creazione di un sito, per chi non lo sapesse, avviene tramite una serie di codici, e linguaggi di programmazione. È quindi molto importante (per lo sviluppatore) mantenere ordine tra le righe e formattare il documento. Per programmare vi consiglio Visual Studio Code (con alcune estensioni). Usare il blocco note o un qualsiasi altro editor di testo non fatto apposta per il coding vi farà diventare pazzo (parlo per esperienza).

Vi consiglio di usare divisori semantici per ordinare il documento HTML. Un esempio sono article, section, header, footer... Un divisore non semantico è il vecchio div.

  • Scegliere il template giusto

Anche questa è abbastanza banale e si collega con il primo consiglio (quello dell'idea). Non è necessario creare un template da capo, come ho fatto io, ma è necessario conoscere un minimo di CSS, in caso qualcosa del template già fatto non vi piaccia.

Potete usare il template del mio sito per farlo incolla la seguente riga di codice tra i tag <head> e </head>:

<link rel="stylesheet" href="https://portalenicola.it/css/core.css">
